# Bookstore Page

A simple, responsive landing page for the XYZ Bookstore. This project displays a collection of featured books with options to view the cart and proceed to checkout.

##  Project Overview

This webpage serves as the storefront for a fictional bookstore. It features:
* **Header Section:** Welcomes visitors to the store.
* **Product Cards:** Displays individual books (e.g., "Sally's SciFi Adventure" and "Dave's Cooking Adventure") with descriptions and purchase buttons.
* **Navigation:** Buttons to view the shopping cart and checkout.

## Technologies Used

* **HTML5:** Structure of the web page.
* **CSS3:** (Intended) Used for styling classes like `.card-container` and `.btn`.

##  How to Run Locally

1.  Clone the repository: https://github.com/Simiyuu/bookstore-page.git

  
2.  Navigate to the project directory:

    cd xyz-bookstore-page
3.  Open `index.html` in your preferred web browser.
